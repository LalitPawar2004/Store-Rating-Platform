# FullStack Rating Platform
Express + MySQL + React

## Quick start
1. Create MySQL DB and run `database/schema.sql`
2. Backend:
   cd backend
   cp .env.example .env  # fill DB credentials and JWT_SECRET
   npm install
   npm run dev
3. Frontend:
   cd frontend
   npm install
   npm run dev

Default admin: email admin@platform.test / password Admin@123 (seed in schema.sql)
